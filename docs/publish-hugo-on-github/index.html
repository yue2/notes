    Publish Hugo on Github #

    objectives #

    1. generate hugo site locally
    2. push generated site - in /public to github
    3. use github page to view the site
    • use git to push to my VPS
    • host on my domain

    work #

    generate site locally #

    To generate the site in hugo, cd in the site root, run hugo, the site will be generated in /public 目录。

    problems:

    1. don’t work if the site folder is in an external disk.
      • move to a local disk solved the problem.
    2. themes don’t render
      • stackoverflow qs looks like problem with the base url
      • changed to github page url and run hugo again. solved the problem.

    push to github #

    in the site public folder, run

    init git
git add .
git remote add origin https://github.com/username/repository.git
git commit -m "commit message"
git push origin main
## other commands
git remote -v ## to check remote
git remote set-url origin https://github.com/username/anotherrepository.git ## to change remote git

    view on github pages #

    In github repo settings, check pages. Set main branch as the page source.
