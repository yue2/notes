    Push local files to NGINX using GIT #

    objectives #

    1. install git in virtual machine
    2. connect local git to NGINX
    3. push local static site to NGINX
    4. configure NGINX to host the site

    1. #

    Under user hugo, create directives for the storing of files

    mkdir var/www/

    install git apt-get install git and add new user to check git push and commit adduser gitter password same as hugo user.

    initiate a git on the system

    mkdir site.git && cd site.git
git --bare init
git update-server-info
chown -R www-data.www-data . 
chmod -R 755 .

    allow firewall

    ufw enable
ufw allow ssh
ufw allow http
ufw status
